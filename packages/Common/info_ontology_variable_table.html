Indices

Same as units – NOTE : currently not protected.

For port variables one will allow to change units and indexing as long as they are not used.

Once they are used, they must be locked. Will be implemented

Equations

A variable can be defined by more than one equation.

Used for various purposes: for basic quantities like temperature as the derivative of internal energy with respect to entropy. But also as result from solving an integral equation from the heat capacity and the computed enthalpy.

Also used to define different behaviours in a different context (liquid, solid etc as an example).

Delete

Computes first what is deleted if the variable disappears, namely all equations that are depending on this variable. Procedure is recursive, but stops at states.
